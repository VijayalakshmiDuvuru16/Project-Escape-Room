# 🔐 Python Escape Room Game

## 📌 Description
This is a console-based escape room game built using Python. 
Players solve puzzles, unlock doors, and try to escape before running out of lives.

## 🎮 Features
- Multiple puzzle doors
- Random lucky number game
- Limited lives system
- Badge rewards

## 🛠️ Technologies Used
- Python
- Random module

## ▶️ How to Run
```bash
python escape_room.py
